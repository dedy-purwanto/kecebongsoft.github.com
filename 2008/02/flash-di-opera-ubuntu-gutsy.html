Flash di Opera (Ubuntu Gutsy)

Salah satu masalah kemarin setelah nginstal Gutsy Tribe (Ubuntu 7.10) adalah opera yang nggak bisa memainkan file swf (shockwave flash), parah banget soalnya aku suka nonton hentai dan miyabi 4 mata di youtube. Tapi tunggu dulu, kenapa aku pakai Opera? secara udah bertahun-tahun pakai firefox. Alasannya nggak lain karena aku pake KDE dan font interface firefox gedenya seanjrit-anjrit segaban-gaban klo di KDE, udah dicoba ubek-ubek konfigurasi tapi tetep aja :-( .

Well, kalau nginstal opera di gutsy dengan perintah apt-get install opera, untuk yang sekarang kita akan dikasih versi 9.25 (versi terbaru yang stabil), kemudian kalau mo pasang flash player, harus nginstall manual. ada beberapa cara : Cara 1 : - Pertama download dulu flash playernya di website Adobe , pilih yang paket tar.gz - Ekstrak ke direktori (desktop contohnya) - Kopi file libflashplayer.so yang ada di folder ekstrak ke folder /usr/lib/opera/plugins - Restart opera, tes dengan membuka halaman ini Cara 2 : - Melalui konsol, masuk ke direktori hasi ekstrak flash player tadi - Login sebagai root, masukan perintah ./flashplayer-installer - Pas ditanya install destination, masukan /usr/lib/opera - Restart opera, tes dengan membuka halaman ini Nggak bisa juga?, coba pake hard way, ikutin tutorial dari forum ubuntu . Dan coba restart opera lalu test apakah flashnya berjalan atau enggak.

Masih nggak bisa juga? pastiin dulu option plugin di opera nyala, tekan F12 ato pilih menu Tools -> Quick Preferences -> Enable Plugin.

Kalau masih nggak bisa, pakai cara terakhir, install opera 9.5 (yang  sekarang masih beta) nggak ada resikonya sih karena sejauh ini aku juga makai nggak ada masalah :p , its run very stable and fast. Download Opera 9.5 Beta disini . Untunglah paketnya debian jadi installnya nggak ribet. And let's say hello to Opera!.
